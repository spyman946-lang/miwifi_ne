 MiWiFi NE — копия интеграции MiWiFi для Home Assistant (отдельный домен) создавалась с целью поддержки роутера AX3000 NE
================================================================================

НАЗНАЧЕНИЕ
----------
Это полная копия компонента «miwifi» с другим идентификатором интеграции:
домен **miwifi_ne**, отображаемое имя **MiWiFi NE**.

Нужна, если вы хотите завести вторую запись интеграции параллельно с «MiWiFi»
(другой роутер, тест, плавный переезд) без конфликта домена `miwifi`.

Код, API LuCI, поведение HTTP/HTTPS при входе и список моделей в enum.py
совпадают с основной папкой **miwifi** на момент копирования; дальнейшие
правки нужно дублировать вручную или синхронизировать самостоятельно.


УСТАНОВКА
---------
1. Папка miwifi_ne должна лежать в:
   <конфигурация HA>/custom_components/miwifi_ne/

2. Перезапуск Home Assistant.

3. Добавить интеграцию → искать **MiWiFi NE** (или miwifi_ne).

Не добавляйте один и тот же роутер одновременно в miwifi и miwifi_ne —
получатся дубли сущностей.


ДОКУМЕНТАЦИЯ
